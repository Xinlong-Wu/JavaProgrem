# [JavaProgram] 图像转化压缩及文件临时存储
>作者：乌鑫龙
---
## 目前想法：
  1. 图片的几种主流格式互相转化 
  2. 利用网上的图像压缩算法将图像进行处理
  
## 需求：

 - 界面   
 [ ] 一个登陆界面
 [ ] 主界面，头部导航条有功能，用户名信息
    中部主窗口将导入的图片显示 

 - 功能：   
 [ ] 利用后台服务器进行注册登陆，服务按月购买制
 [ ] 图片以类的形式展现
